# Lab · Phase 1 Week 6 (Calendar week 6)
## Git worktrees & parallel AI experiments

**Codebase:** [Canvas LMS](https://github.com/instructure/canvas-lms) · [`../../brownfield-project.md`](../../brownfield-project.md)

### Goal
Run **two parallel explorations** of the same **Canvas**-scoped task in separate worktrees.

### Steps

1. Define a bounded spike (e.g., “where should a change to X live?” across `app/`, `ui/`, or `packages/`).
2. Spin two worktrees; use different prompt strategies or tool setups.
3. Have AI summarize diff-style pros/cons; you decide if either is merge-worthy.
4. Document commands you used and cleanup steps.

### Definition of done
Short decision record: pursue / park / reject with reasons.
