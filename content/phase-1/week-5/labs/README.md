# Lab · Phase 1 Week 5 (Calendar week 5)
## Model Context Protocol (MCP) for Canvas LMS navigation

**Codebase:** [Canvas LMS](https://github.com/instructure/canvas-lms) · [`../../brownfield-project.md`](../../brownfield-project.md)

### Goal
Complete one **real Canvas LMS investigation** faster using MCP-backed tools than without.

### Steps

1. Pick a question that spans multiple directories or doc types.
2. Run the investigation twice: baseline vs MCP-assisted (time-box 30 min each).
3. Log prompts, tool calls (redacted), and outputs.
4. Write a half-page retrospective: what was genuinely better vs theater?

### Definition of done
Comparison table + one concrete improvement to your workflow.
