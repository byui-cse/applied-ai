# Lab · Phase 1 Week 1 (Calendar week 1)
## Context packs & markdown as the UI for AI

**Codebase:** [Canvas LMS](https://github.com/instructure/canvas-lms) · [`../../brownfield-project.md`](../../brownfield-project.md)

### Goal
Produce a **Context Pack v0** for **Canvas LMS** entirely through AI dialogue.

### Steps

1. Prompt the model to infer the repo's purpose from top-level files and docs; fix mistakes by *pointing* to paths, not pasting whole files.
2. Ask for a **system map**: entrypoints, configs, data boundaries, and top risks.
3. Iteratively **compress**: remove duplication; keep glossary and invariants; target a lean pack you could paste at the start of a session.
4. Optional: have AI propose a **folder-index** table (path → responsibility) without dumping file contents.

### Definition of done
Pack is <800 words OR fits your self-imposed context slice; a peer can orient in 5 minutes.
