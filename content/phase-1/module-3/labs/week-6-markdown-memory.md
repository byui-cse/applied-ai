# Week 6 Lab: Markdown Memory (Worktree Plan + Diff Summary)

## `worktree-plan.md` template
```markdown
# Worktree Plan

## Main branch strategy
<what must remain stable>

## Worktree branch strategy
<what experiments are isolated here>

## Verification checklist
- <tests/linters/builds you will run>

## Exact git workflow (as text)
1. <create worktree command>
2. <make changes>
3. <run tests>
4. <record result>
```

## Diff summary template
```markdown
Change summary (high level)
- <what you changed and why>

What to verify
- <tests/checks>

Risk notes / rollback plan
- <what might fail and what you would do next>
```

## Deliverables
- `worktree-plan.md` (with commands + verification checklist)
- Diff summary template (2-3 paragraphs)
