# Week 7 Lab: Agents (Studio Workflow Protocol)

## Agent workflow (end-to-end)
1. Map
   - Build your system map from the smallest necessary context
2. Plan
   - Produce a task plan with explicit assumptions
3. Evidence + Tools
   - Gather evidence and verify using your tool plan
4. Propose changes (bounded)
   - Reference:
     - your context budget
     - your architectural memory artifacts
     - your caching/debt workflow
5. Verify + record
   - Run required tests/checks
   - Record results and update artifacts if needed

## Iteration protocol (two runs)
1. Iteration 1 (conservative)
   - Minimal scope and simple evidence collection
   - Draft the “handoff artifact” with basic verification steps
2. Iteration 2 (refine)
   - Tighten tool plan and verification steps
   - Strengthen the risk plan and reduce token usage by reusing artifacts

## Submission checklist
- `week-7-context-management.md`
- `week-7-markdown-memory.md`
- A short reflection: one insight and one thing to improve next time

## Reflection
- At what point did you spend the most tokens, and how will you reduce it later?
- What verification step gave you the most confidence?
