# Week 3 Lab: Markdown Memory (AI Memory Artifacts)

## Deliverable templates

### `architecture-summary.md`
```markdown
# Architecture Summary

## What the system is
<one-paragraph overview>

## Major pieces and how they connect
1. <component / module name>
   - Responsibilities:
   - Inputs / outputs:
   - Key interactions:

## Boundaries
<what is inside vs outside this summary?>
```

### `key-decisions.md` (decision log)
```markdown
# Key Decisions

## <YYYY-MM-DD> - <decision title>
- Decision:
- Why:
- Evidence / links:
- Expected impact:
```
