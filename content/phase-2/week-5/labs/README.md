# Lab · Phase 2 Week 5 (Calendar week 12)
## Single-agent systems for specialized dev tasks

### Goal
Automate (via agent) one **bounded** dev chore end-to-end with checkpoints.

### Steps

1. Define inputs, outputs, and forbidden actions.
2. Run the agent; collect artifacts.
3. Post-mortem: where it went wrong and which rule you'd add.

### Definition of done
Repeatable prompt + checklist that works on a second attempt.
