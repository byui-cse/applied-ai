# Lab · Phase 2 Week 2 (Calendar week 9)
## Skills, rules, and repeatable greenfield rituals

### Goal
Author a **Rules/Skills Pack** tailored to your greenfield MVP.

### Steps

1. List 5 recurring tasks AI will do (e.g., add endpoint, add component).
2. For each, write a short rule: inputs, outputs, constraints, verify steps.
3. Have AI critique its own rules—then you adjudicate.
4. Try one real task end-to-end using only the pack + prompts.

### Definition of done
Pack fits on ~2–3 screens; task completes with fewer follow-up corrections than baseline.
