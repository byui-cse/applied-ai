# Week 8 Lab: Agents (Two-Iteration Docs Workflow)

## Agent workflow (for yourself)
1. Produce the handbook artifacts
   - `ai-handbook.md`
   - `acceptance-criteria.md`
2. Create a test plan section that makes acceptance criteria executable
3. Write a prompt contract that tells a future AI exactly how to use the handbook and what to verify

## Iteration protocol (two runs)
1. Iteration 1 (conservative)
   - Draft handbook structure + acceptance criteria + test plan
   - Draft prompt contract that references those sections
2. Iteration 2 (tighten for reuse)
   - Ensure acceptance criteria are concrete and testable
   - Make the “how to run” and “how to add a feature” checklists specific
   - Simplify prompt contract to remove anything not required for correct outputs
