# Week 9 Lab: Agents (Dry Evaluation Loop)

## Agent workflow
1. Build the suite (what inputs you will provide, what outputs you require)
2. Define the acceptance rubric (what must be true)
3. Run one dry evaluation (score one hypothetical response)
4. Iterate: tighten rubric and suite to prevent context drift

## Iteration protocol
1. Iteration 1 (conservative)
   - Choose 3 tasks
   - Write the suite + rubric
   - Score one dry evaluation
2. Iteration 2 (refine)
   - Improve criteria to be more checkable
   - Adjust output format to make acceptance properties easier to observe
