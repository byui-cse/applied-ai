# Lab · Phase 2 Week 3 (Calendar week 10)
## Plugins & extensibility: extending the LLM in-app

### Goal
Draft a **plugin or tool integration** spec for your MVP.

### Steps

1. Describe user-visible behavior in 5 bullets.
2. AI generates OpenAPI-like or JSON-schema descriptions; you validate against reality.
3. List abuse cases; add guardrails.
4. Outline a test plan *executed by AI* with human approval gates.

### Definition of done
Another team could implement from spec without guessing your intent.
