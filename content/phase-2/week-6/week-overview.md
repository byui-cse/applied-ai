**Module:** 6 — Autonomous agents  

## Learning outcomes

By the end of this week, students should be able to:

1. **Justify** when multi-agent separation (research vs implementation vs review) beats a single long thread.
2. **Compare** shared-artifact workflows versus message-passing between agents.
3. **Resolve** or escalate conflicts when two agent outputs disagree, using explicit criteria.
4. **Contrast** orchestrator-led multi-agent setups with peer-style agent topologies.
5. **Mitigate** stalls and circular debates with timeouts, escalation paths, and least-privilege roles per agent.
6. **Produce** a **multi-agent protocol**: roles, handoff artifacts, and communication rules for their MVP workflow.
