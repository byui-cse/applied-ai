# Lab · Phase 2 Week 6 (Calendar week 13)
## Multi-agent coordination & handoffs

### Goal
Execute a **two-agent** workflow on one MVP task (e.g., spec agent → builder agent).

### Steps

1. Write a handoff template: state, assumptions, open questions, next commands.
2. Run agent A; freeze its output; feed to agent B without re-explaining the world.
3. Measure rework: how many corrections were needed?

### Definition of done
Handoff template reusable for a second task with <30% edit overhead.
