Week 8: Foundations & Skills

Build AI-Ready Architecture from Scratch

Learning objectives

  • Scaffold a repository so LLM workflows are easier and safer
  • Create documentation and test scaffolding that reduces prompt ambiguity
  • Establish conventions for future tool integrations (plugins and agents)

Key ideas

  • AI-ready architecture includes structure for: docs, examples, tests, and stable interfaces
  • Consistency beats novelty: predictable folder patterns reduce cognitive load
  • Good evaluation starts with acceptance criteria you can test

  • Week 8 Context Management: AI-Ready Docs + Tests
  • Week 8 Markdown Memory: AI Handbook + Prompt Contract
