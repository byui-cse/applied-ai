## Design Context

### Users
Students (and emerging software engineers) using an applied AI course site to learn practical workflows for day-to-day development tasks.

### Brand Personality
Editorial, warm, crafted.

### Aesthetic Direction
Minimal layout with strong readability and clear section rhythm. Support both light and dark themes with consistent contrast. Subtle motion only (no distraction).

### Design Principles
1. Clarity-first hierarchy: spacing and type scale do most of the work; avoid visual clutter.
2. Comfortable reading experience: stable line length (measure-limited) and generous line-height for long passages.
3. Accessibility by default: keyboard usability, visible focus, adequate target sizing, and strong contrast across themes.
4. Editorial rhythm: tight grouping within sections and distinct separation between major blocks to guide scanning.
5. Theming consistency: prefer CSS tokens/variables over hard-coded colors so both themes stay coherent.
