Applied AI for Software Development

A 14-week applied course for real-world engineering work

Course overview

This course teaches applied AI skills for real software engineering work, with a focus on keeping LLM interactions efficient, safe, and useful inside day-to-day development workflows.

You will work across two modes:

  • Brownfield (Legacy Systems): refactor and modernize existing codebases without blowing up context limits or losing architectural intent.
  • Greenfield (New Systems): design clean, AI-ready architectures from scratch and extend them with plugins and autonomous agent workflows.

What you will practice

  • Designing prompts and workflows that stay context-efficient (including token budgeting)
  • Managing "architectural state" across iterations (memory, caching, and debt tracking)
  • Using tools effectively (MCP + safe multi-branch refactoring strategies)
  • Building agent and multi-agent systems for development tasks
  • Orchestrating complex workflows to ship a production-ready MVP
